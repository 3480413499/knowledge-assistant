# 知识小助手 (Knowledge Assistant)

RAG 智能知识库问答系统 — 上传你的 PDF/Word/TXT 文档，用自然语言提问，AI 精准回答并标注来源。

## 技术栈

- **前端**：Streamlit
- **LLM**：DeepSeek API
- **嵌入模型**：BGE-M3（本地部署）
- **向量数据库**：ChromaDB
- **文档解析**：PyMuPDF

## 快速开始

```bash
pip install -r requirements.txt
streamlit run app.py
```
